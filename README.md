# Gerenciador de Tarefas

Um sistema simples de gerenciamento de tarefas construído com Next.js e tRPC.

## Funcionalidades

- Listagem de tarefas
- Criação de novas tarefas
- Edição de tarefas existentes
- Exclusão de tarefas
- Validação de formulários

## Tecnologias utilizadas

- Next.js 15
- tRPC
- TypeScript
- Tailwind CSS
- Zod (para validação)
